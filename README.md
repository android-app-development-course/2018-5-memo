## 应用名称：Memo
## 应用图标：  

<image src="./系统文档/picture/icon.png" width="150px"/>    

这是一个简单的便签app，具备基本的编辑和保存功能；  
### 基本功能介绍
长按应用主界面右上角图标可进入隐私模式；   
左/右滑动便签可进行删除操作。
